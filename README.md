# Barber Queue MVP
مشروع أولي لحجز مواعيد الحلاقين باستخدام Next.js + Convex + Tailwind.

## تشغيل المشروع
1. إنشاء حساب Convex وأخذ رابط الـ Deployment.
2. إعداد متغير البيئة:
   NEXT_PUBLIC_CONVEX_URL=<رابط Convex الخاص بك>
3. تشغيل المشروع:
   npm install
   npm run dev
4. زيارة http://localhost:3000
